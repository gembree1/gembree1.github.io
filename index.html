About Visualization

EDIT The project follows the interactive slide show narrative visualization structure. The Home page represents the base of the visualization structure. The user is being exposed to the author driven content and will be led through three different scenes/slides with giving user ability to move backwards and forward as well as jumping directly to specfic slide by clicking on top navigation bar.

Rubric

  • Does the narrative visualization correctly follow the hybrid structure as stated by the essay?

    • Interactive slide show narrative visualization structure is followed.

  • Does the narrative visualization effectively utilize scenes?

    • Four different slides are used to give reader various perspective. Scenes are primilarily Author driven over mouse scroll event by user. However, Reader can jump to random scenes if he/she prefers.

  • Does the narrative visualization effectively utilize annotations?

    • Tooltip, change of colors and Axes names etc. are used as annotations for different scenes.

  • Does the narrative visualization effectively utilize parameters?

    • Parameters are passed when user choose to jump to random scenes by clicking on navigation bar. Parameters are also used to give user additional details during mouse hover event by showing a tooltip.
    e.g. Breweries By population
    Bar chart uses mouse position as parameter to mousehover event to display tooltip and chnage the other parameter color value.

  • Does the narrative visualization effectively utilize triggers?

    • The mouse click on navigation bar acts as a trigger event for navigation to different slides.
